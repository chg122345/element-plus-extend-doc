ExTreeSelect 树选择器 ​

TIP

类似 Select 的信息录入控件，拥有和 Select 几乎相同的 API，适用于选择树形的数据结构

基础用法 ​

选项动态化 ​

在 props.formatter的情况下显示值,此配置不改变绑定的值

WARNING

如需改变绑定的值，请使用valueFormat 和 valueParser组合

API ​

Props ​

名称类型默认值说明必传
model-value / v-modelArray/string-选中项绑定值Y
placeholderstring-占位文本N
clearablebooleanfalse是否可清空N
disabledbooleanfalse是否被禁用N
treePropsobject-树组件属性，参照ElTreeV2N
multiplebooleanfalse是否多选N
multipleLimitnumber-多选数量限制N
autoSelectFirstbooleanfalse自动选中第一个N
autoClearMismatchbooleanfalse自动清除不匹配的值N
valueTypevalue objectvalue绑定值类型N
optionsArray-选项数据N
propsobject{label: 'label',value: 'value', children: 'children'}选项数据的配置 请参阅下面OptionPropsN
loadImmediatebooleantrueapi是否主动加载数据，为false时懒加载远程搜索N
apiFunction-选项数据加载apiN
beforeApiFetchFunction/object-请求api前调用返回值用于api携带参数N
parseApiDataFunction-解析api返回的数据N
apiEffectKeyFunction/string-动态加载apikey，值变化后会调用组件reload方法N
valueFormatFunction-绑定值格式化函数N
valueParserFunction-反解析valueFormatN

OptionProps ​

名称类型默认值说明
labelstringlabel指定选项标签为选项对象的某个属性值
valuestringvalue指定选项的值为选项对象的某个属性值
childrenstringchildren指定选项的子选项为选项对象的某个属性值
disabledstring/Functiondisabled指定选项的禁用为选项对象的某个属性值/返回是否禁用
showstring/Function-指定选项的是否显示为选项对象的某个属性值/返回是否显示
formatterFunction-选项显示格式化函数

Events ​

名称参数说明
update:modelValue(value)值改变时触发，value为选中项绑定值
change(value, {value, trigger})值改变时触发，value为选中项绑定值，trigger为触发类型：initial、change、clearMismatch、autoSelect
change-data(data, {trigger})值改变时触发，data为选中选项的值，trigger为触发类型：initial、change、clearMismatch、autoSelect
loaded-options(options)绑定api时数据加载完成触发，options为api返回的选项数据

Slots ​

名称参数说明
default({data, node})自定义备选项的节点内容，data为选项数据，node选项树节点
header-下拉面板顶部内容
footer-下拉面板底部内容
empty-无选项时内容

Exposes ​

名称类型说明
reload(params?:Record<string, unknown>)调用api重新加载选项数据，params为调用api携带的参数
bindOptionsData(data: Array<IOptionData>)重新绑定组件选项数据，data为选项数据
