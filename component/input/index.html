ExInput 输入框 ​

WARNING

Input 为受控组件，它 总会显示 Vue 绑定值。

不支持 v-model 修饰符。

基础用法 ​

搜索
查看源代码
vue
<template>
  <ex-input v-model="modelValue" :slots="{append: '搜索'}"></ex-input>
</template>
<script lang="ts" setup>
import {ExInput} from "element-plus-ex";
import {ref} from "vue";
const modelValue = ref();
</script>

数字输入 ​

通过 type 属性指定值为 number

保留2为小数
显示千分位
数字右对齐
查看源代码
vue
<template>
  <ex-input v-model="modelValue" type="number"></ex-input>
  <ex-input v-model="modelValue2" type="number" :precision="2"></ex-input>
  <ex-input v-model="modelValue2" type="number" :precision="2" :min="0.88" :max="10000.88" :to-thousand="true"></ex-input>
  <ex-input v-model="modelValue2" type="number" :precision="2" :to-thousand="true" text-position="right"></ex-input>
</template>
<script lang="ts" setup>
import {ExInput} from "element-plus-ex";
import {ref} from "vue";
const modelValue = ref();
const modelValue2 = ref();
</script>

手机号输入 ​

通过 type 属性指定值为 phone

查看源代码
vue
<template>
  <ex-input v-model="modelValue" type="phone"></ex-input>
</template>
<script lang="ts" setup>
import {ExInput} from "element-plus-ex";

格式化显示 ​

在 formatter的情况下显示值,此配置不改变绑定的值

WARNING

如需改变绑定的值，请使用valueFormat 和 valueParser

查看源代码
vue
<template>
  <ex-input v-model="modelValue" :formatter="(value) => '前缀-' + value"></ex-input>
</template>
<script lang="ts" setup>
import {ExInput} from "element-plus-ex";
import {ref} from "vue";
const modelValue = ref();
</script>

API ​

Props ​

名称类型默认值说明必传
model-value / v-modelstring/number-绑定值Y
typetext textarea number phone 等text类型N
maxlengthstring/number-最大输入长度N
minlengthstring/number-最小输入长度N
showWordLimitbooleanfalse是否显示统计字数N
disabledbooleanfalse是否被禁用N
readonlybooleanfalse是否只读N
placeholderstring-占位文本N
clearableboolean-是否可清空N
showPasswordbooleanfalse是否显示切换密码图标N
sizestring-组件大小N
prefixIconstring-自定义前缀图标N
suffixIconstring-自定义后缀图标N
rowsnumber-输入框行数，仅 type 为 'textarea' 时有效N
autosizeboolean/objectfalsetextarea 高度是否自适应N
resizenone both horizontal vertical-控制是否能被用户缩放N
precisionnumber-数值精度，type=number时有效N
maxnumber-设置最大值N
minnumber-设置最小值N
minusAbleboolean-是否可输入负数，type=number时有效N
toThousandboolean-是否显示成千分符，type=number时有效N
textPositionleft center right-数值对齐方式，type=number时有效N
formatterFunction-格式化显示，不改变绑定值N
valueFormatFunction-绑定值格式化函数N
valueParserFunction-反解析valueFormatN

Events ​

名称参数说明
update:modelValue(value)值改变时触发，value为绑定值
input(value)值改变时触发，value为绑定值
blur-失去焦点时触发
focus-获得焦点时触发

Slots ​

名称参数说明
prefix-输入框头部内容
suffix-输入框尾部内容
prepend-输入框前置内容，在 prefix 之前
append-输入框后置内容，在 suffix 之后
