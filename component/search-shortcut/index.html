ExSearchShortcut 快捷搜索 ​

基础用法 ​

查看源代码
vue
<template>
  <ex-search-shortcut v-bind="searchFormProps" />
  <span class="example-title">圆角样式</span>
  <ex-search-shortcut v-bind="searchFormProps" round />
</template>

<script lang="ts" setup>
import {IExSearchShortcutSchema} from "element-plus-extension";
import {ref} from "vue";

const searchFormProps = ref({
  schemas: [
    {
      prop: 'name',
      label: '姓名',
    },
    {
      prop: 'name1',
      label: '姓名1',
    },
    {
      prop: 'name2',
      label: '姓名2',
    },
    {
      prop: 'age',
      label: '年龄',
    },
    {
      prop: 'age1',
      label: '年龄1',
    },
    {
      prop: 'age2',
      label: '年龄2',
    },
    {
      prop: 'birthday',
      label: '生日',
    },
    {
      label: '兴趣爱好',
      prop: 'hobby',
    },
    {
      label: '邮箱',
      prop: 'email',
    },
  ] as IExSearchShortcutSchema[]
});

</script>

API ​

Props ​

名称类型默认值说明必传
schemasArray-字段选项配置，数据类型请参阅下面IExSearchShortcutSchemaY
placeholderstring-占位文本N
clearablebooleantrue是否可清空N
sizestring-组件大小N
roundbooleanfalse是否为圆角N
maxnumber10最大选择字段数N
defaultCheckedFieldsArray-默认选中的字段N

Events ​

名称参数说明
search(value)按下键盘Enter键时触发，value搜索值，数据类型请参阅下面IExSearchShortcutSchema
reset(value)输入框清空值时触发

Exposes ​

名称类型说明
getSearchValue() => IExSearchShortcutModel获取搜索值
setSearchValue(value: IExSearchShortcutModel)设置搜索值

类型声明 ​

IExSearchShortcutSchema ​

ts
export interface IExSearchShortcutSchema {
  prop: string;
  label: string;
}

IExSearchShortcutModel ​

ts
export interface IExSearchShortcutModel {
  /** 勾选的需要搜索的字段*/
  fields: string[],
  /** 是否勾选了搜索全部*/
  isSearchAllFields: boolean,
  /** 输入框输入的值*/
  searchValue: string;
}
