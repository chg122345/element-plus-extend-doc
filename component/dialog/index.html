ExDialog 对话框 ​

Ref用法 ​

绑定Ref属性调用open方法打开对话框

查看源代码
vue
<template>
  <el-button @click="handleOpen">点击打开模态框</el-button>
  <el-button @click="dialogRef2.open()">点击打开模态框-隐藏底部操作区</el-button>
  <el-button @click="dialogRef3.open()">点击打开模态框-自定义头部</el-button>

  <ex-dialog ref="dialogRef" title="弹框标题">
    弹框内容
  </ex-dialog>
  <ex-dialog ref="dialogRef2" title="弹框标题" :with-footer="false" :height="400">
    弹框内容,隐藏底部操作区
  </ex-dialog>
  <ex-dialog ref="dialogRef3" title="弹框标题" :height="300">
    <template #header>
      自定义头部 <ex-input style="width: 180px" clearable />
    </template>
    自定义头部, 设置的标题属性被覆盖
  </ex-dialog>
</template>

<script lang="ts" setup>

import {ref} from "vue";
import {IExDialogInstance} from "@element-plus-extension";

const dialogRef = ref<IExDialogInstance>();
const dialogRef2 = ref<IExDialogInstance>();
const dialogRef3 = ref<IExDialogInstance>();

const handleOpen = () => {
  dialogRef.value.open();
}
</script>

useDialog注册绑定方式用法 ​

TIP

对于复杂的场景，可以配合 useDialog 来使用

查看源代码
vue
<template>
  <el-button @click="handleOpen">点击打开模态框-设置加载状态</el-button>

  <ex-dialog @register="register" @confirm="handleConfirm">
    弹框内容，点击确认按钮后触发确认事件
  </ex-dialog>
</template>

<script lang="ts" setup>
import {useDialog} from "@element-plus-extension";
const [register, {open, close, setProps}] = useDialog({
  title: '弹框测试',
  width: 500,
  height: 500,
});
const handleOpen = () => {
  setProps({
    loading: true
  })
  open();
  setTimeout(() => {
    setProps({
      loading: false
    })
  }, 3000)
}
const handleConfirm = () =>  {
  setProps({
    confirmLoading: true
  })
  setTimeout(() => {
    setProps({
      confirmLoading: false
    });
    // 关闭弹窗
    close();
  }, 3000)
}
</script>

API ​

Props ​

名称类型默认值说明必传
titlestring-标题N
widthstring/number50%宽度N
heightstring/number-高度N
maxHeightstring/number-最大高度N
fullscreenbooleanfalse是否为全屏N
topstring15vh距顶部距离N
modalbooleantrue是否需要遮罩层N
modalClassstring-遮罩的自定义类名N
teleportedbooleantrue自身是否插入至 body 元素上，和appendToBody相同N
lockScrollbooleantrue是否将body滚动锁定N
openDelaynumber0打开的延时时间，单位毫秒N
closeDelaynumber0关闭的延时时间，单位毫秒N
closeOnClickModalbooleanfalse是否可以通过点击 modal 关闭N
closeOnPressEscapebooleantrue是否可以通过按下 ESC 关闭N
showClosebooleantrue是否显示关闭按钮N
beforeCloseFunction-关闭前的回调， 回调函数内执行done参数方法的时候才会关闭N
draggablebooleantrue是否启用可拖拽功能N
centerbooleanfalse是否居中排列N
destroyOnClosebooleantrue关闭时，销毁其中的元素N
zIndexnumber-和原生的CSS的z-index相同N
lazybooleantrue延时加载（首次打开弹窗才去渲染）N
withHeaderbooleantrue是否显示头部N
withFooterbooleantrue是否显示底部N
showZoombooleantrue是否显示缩放全屏N
loadingbooleanfalse加载中动画N
showConfirmButtonbooleantrue是否显示确认按钮N
showCancelButtonbooleantrue是否显示取消按钮N
confirmButtonTextstring确认确认按钮文本N
cancelButtonTextstring取消取消按钮文本N
confirmLoadingbooleanfalse确认按钮加载中动画N

Events ​

名称参数说明
exit-点击关闭按钮时触发，同时关闭弹窗
cancel-点击取消按钮时触发，同时关闭弹窗
confirm-点击确认按钮时触发
zoom({fullscreen: boolean})点击全屏缩放时触发
register(instance)注册事件

Slots ​

名称参数说明
header-顶部操作区，会替换标题部分，但不会移除关闭按钮。
default-自定义内容
footer-底部操作区，会替换确认 取消 按钮

Exposes ​

名称类型说明
setProps(data?:Record<string, unknown>)设置属性方法，data为需要设置对话框的props值
open(data?:Record<string, unknown>)打开对话框方法，data为自定义设置的参数
close-关闭对话框方法

Hooks ​

useDialog ​

ts
const [register, {...methods}] = useDialog(data);
名称类型说明
useDialog(data?:Record<string, unknown>)data为需要设置对话框的props值
registerFunctionuseDialog返回的处理注册的方法，绑定到register事件上
methods-useDialog返回的组件实例方法，详见Exposes
