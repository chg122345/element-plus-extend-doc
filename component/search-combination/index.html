API ​

Props ​

名称类型默认值说明必传
schemasArray-字段选项配置，数据类型请参阅IExFormSchemaY
modelValueArray/object-绑定值， 数据类型请参阅下面IExSearchCombinationModelN
multiplebooleanfalse是否多条件组合N

Events ​

名称参数说明
update:modelValue(value)点击确认搜索时触发，value搜索值，数据类型请参阅下面IExSearchCombinationModel
search(value)点击确认搜索时触发
reset(value)点击重置或点击清空时触发

Slots ​

名称参数说明
trigger()自定义节点内容，data为节点数据，node为树节点

Exposes ​

名称类型说明
getSearchValue() => Array<IExSearchCombinationModel>获取搜索值
setSearchValue(value: Array<IExSearchCombinationModel>)设置搜索值

组合搜索面板API ​

Props ​

名称类型默认值说明必传

Events ​

名称参数说明

Exposes ​

名称类型说明
getSearchValue() => IExSearchCombinationPaneModel获取搜索值
setSearchValue(value: IExSearchCombinationPaneModel)设置搜索值

类型声明 ​

IExSearchCombinationModel ​

ts
export interface IExSearchCombinationModel {
  /** 搜索条件字段*/
  field: string;
  /** 比较符*/
  comparison: string;
  /** 条件值*/
  value: IModelValueArray | IModelValue;
  /** 字段类型*/
  type?: string;
}

IExSearchCombinationPaneModel ​

ts
export interface IExSearchCombinationPaneModel {
  join: 'AND' | 'OR';
  children: Array<IExSearchCombinationModel | IExSearchCombinationPaneModel>;
}
