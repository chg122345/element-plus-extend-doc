API ​

Props ​

名称类型默认值说明必传
model-value / v-modelArray/object-选中项绑定值Y
placeholderstring-占位文本N
clearablebooleanfalse是否可清空N
disabledbooleanfalse是否被禁用N
readonlybooleanfalse是否只读N
autosizebooleanfalse高度是否自适应N
placeholderstring-占位文本N
clearableboolean-是否可清空N
filterablebooleantrue是否可过滤N
filterNodeMethodFunction-自定义过滤方法，方法参数(value, data)，value为搜索关键字，data为选项数据N
sizestring-组件大小N
prefixIconstring-自定义前缀图标N
suffixIconstring-自定义后缀图标N
multiplebooleanfalse是否多选N
multipleLimitnumber-多选数量限制N
optionsArray-选项数据，数据类型请参阅下面IExOrgOptionsDataN
propsobject{label: 'label',value: 'value'}选项数据的配置 请参阅下面OptionPropsN
loadImmediatebooleantrueapi是否主动加载数据N
apiFunction-选项数据加载apiN
beforeApiFetchFunction/object-请求api前调用返回值用于api携带参数N
parseApiDataFunction-解析api返回的数据N
apiEffectKeyFunction/string-动态加载apikey，值变化后会调用组件reload方法N
valueFormatFunction-绑定值格式化函数N
valueParserFunction-反解析valueFormatN

OptionProps ​

名称类型默认值说明
labelstringlabel指定选项标签为选项对象的某个属性值
valuestringvalue指定选项的值为选项对象的某个属性值
childrenstringchildren指定选项的子选项为选项对象的某个属性值
disabledstring/Functiondisabled指定选项的禁用为选项对象的某个属性值/返回是否禁用
showstring/Function-指定选项的是否显示为选项对象的某个属性值/返回是否显示
formatterFunction-选项显示格式化函数
showCheckboxFunction-返回选项是否显示勾选框

Events ​

名称参数说明
update:modelValue(value)值改变时触发，value为选中项绑定值
change(value, {value, trigger})值改变时触发，value为选中项绑定值，trigger为触发类型：initial、change
change-data(data, {trigger})值改变时触发，data为选中选项的值，trigger为触发类型：initial、change
loaded-options(options)绑定api时数据加载完成触发，options为api返回的选项数据

Slots ​

名称参数说明
default({data})自定义备选项的节点内容，data为选项数据
valueDisplay({value})自定义绑定值内容， value为选项绑定值
prefix-输入框头部内容
suffix-输入框尾部内容
trigger({trigger, disabled, size})自定义触发插槽，trigger打开弹框的方法

Exposes ​

名称类型说明
reload(params?:Record<string, unknown>)调用api重新加载选项数据，params为调用api携带的参数
bindOptionsData(data: Array<IExOrgOptionsData>)重新绑定组件选项数据，data为选项数据
