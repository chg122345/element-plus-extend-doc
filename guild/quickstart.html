快速开始 ​

本节将介绍如何在项目中使用组件库element-plus-extension

按需引入 ​

vue
<template>
  <ExInput></ExInput>
</template>

<script setup>

import { ExInput } from 'element-plus-extension';

</script>

全局引入 ​

main.ts

typescript
import {createApp} from 'vue';
import ElementPlusExtension from 'element-plus-extension';
import 'element-plus-extension/dist/theme-chalk/index.css';
import App from './App.vue'
const app = createApp(App);
app.use(ElementPlusExtension);
app.mount('#app')
